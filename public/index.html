URL shortener

Redirections count

GET /api/count/{urlKey}

Return the redirections count for the specified url key. It may not work if the shortned URL is http only (no https)

urlKey: string
in path

Short URL key used to delete the original website from the DB.

200 OK
RedirectionsCountResponse

Return the redirection count

404 Not Found

URL key not found

500 Internal Server Error

Internal Server Error

Response Content-Types: application/json
Response Example (200 OK) 
{
  "redirectionsCount": 1
}

Schema Definitions

LocationResponse: object

Contains the shorten URL and the its key

urlKey: string

Short URL key. Use it to access the full url or to delete it

location: string

Full path of the shorten url. It's equal to the Location header

message: string

Error message

Example
{
  "urlKey": "MTU5NTE3Nzk5NjYwNw",
  "location": "http://localhost:8080/api/MTU5NTE3Nzk5NjYwNw",
  "message": "URL already exists in the DB"
}

RedirectionsCountResponse: object

Contains the shorten URL and the its key

redirectionsCount: integer

How many time the short url has been used for redirecting to the original URL

Example
{
  "redirectionsCount": 1
}
